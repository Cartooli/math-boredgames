What is Legendre Symbol?

College Formula Available

For prime p and integer a, (a/p) = 0 if p|a, +1 if a is a quadratic residue mod p, −1 otherwise. Multiplicative in a.. The key formula is (a/p) = a^((p−1)/2) mod p. This concept is typically introduced in College. Understanding this concept builds a strong foundation for more advanced mathematics.

Key Formula

(a/p) = a^((p−1)/2) mod p
