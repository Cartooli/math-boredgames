What is Quadratic Residue?

College Formula Available

a is a quadratic residue modulo n if there exists x with x² ≡ a (mod n). The Legendre symbol (a/p) indicates whether a is a quadratic residue mod prime p.. The key formula is (a/p) = 0, 1, or −1. This concept is typically introduced in College. Understanding this concept builds a strong foundation for more advanced mathematics.

Key Formula

(a/p) = 0, 1, or −1
