üìö What Is Linear?

A linear relationship has a constant rate of change. For every increase in x, y changes by the same amount!

Example: Earning $10 per hour
1 hour ‚Üí $10
2 hours ‚Üí $20
3 hours ‚Üí $30
Constant rate: +$10 per hour ‚úì

üìä Linear Functions

Linear functions have the form: f(x) = mx + b

‚Ä¢ m = slope (rate of change)
‚Ä¢ b = y-intercept (starting value)
‚Ä¢ Graph is always a straight line

üéØ Recognizing Linear Patterns

Check if it's linear:

‚Ä¢ Does it graph as a straight line?
‚Ä¢ Is there a constant rate?

Problem 1: Is this linear? x: 1,2,3,4 and y: 3,5,7,9
