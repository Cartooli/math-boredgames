Numbers Tell Us "How Many"

A number is a symbol that tells us how many of something we have. When you count your fingers (1, 2, 3, 4, 5...), you're using numbers!

Counting: The Foundation

When we count, we say the numbers in order: 1, 2, 3, 4, 5... and so on. Each number tells us how many objects we've counted so far.

âœ“ Pro Tip: When counting, touch or point to each object as you say the number. This helps you not miss any or count the same one twice!
