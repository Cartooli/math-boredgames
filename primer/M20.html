üìö What Is a Variable?

Variables: Symbols for Unknown Values

A variable is a letter or symbol that represents a number we don't know yet‚Äîor a number that can change. Instead of saying "some unknown number," we can just write x or n or any letter we choose!

üìä Common Variables You'll See

x ‚Äî Most common variable (like "x marks the spot")
y ‚Äî Often used with x in graphs
n ‚Äî Short for "number"
t ‚Äî Often represents time
a, b, c ‚Äî Used for coefficients or constants

üí° You can use almost any letter! Mathematicians usually stick to certain letters for certain things.

Expressions: Math with Variables

An expression is a mathematical phrase that can include numbers, variables, and operations (like + or √ó). For example: x + 5 is an expression.

With Numbers:
7 + 5 = 12
We know everything!
With Variables:
x + 5
Depends on what x is!

Reading Variable Expressions

Learn how to read common expressions:

x + 3 ‚Üí "x plus 3" or "3 more than x"
n - 7 ‚Üí "n minus 7" or "7 less than n"
5y ‚Üí "5 times y" or "5y" (we can skip the √ó symbol!)
t √∑ 2 or t/2 ‚Üí "t divided by 2" or "t over 2"
‚úì Pro Tip: When a number is right next to a variable with no symbol between them (like 5x), it means multiplication! This is special notation for algebra. 5x means 5 √ó x.
‚ö†Ô∏è Common Mistake: Don't confuse x (the variable) with √ó (multiplication)! In algebra, we usually write multiplication as ¬∑ or just put numbers and letters together (like 3x).

üí° Let's Use Variables!

Example 1: Translating Words to Expressions

Problem: Write an expression for "8 more than a number n"

Step 1: Identify the variable ‚Üí n (the unknown number)
Step 2: Identify the operation ‚Üí "more than" means addition
Step 3: Write it in math notation ‚Üí n + 8

Answer: n + 8

üí° Key Insight: "More than" and "plus" both mean addition (+)

Example 2: Evaluating Expressions

Problem: If x = 4, what is the value of 3x + 2?

Step 1: Replace x with 4 ‚Üí 3(4) + 2
Step 2: Multiply first ‚Üí 12 + 2
Step 3: Add ‚Üí 14

Answer: 14

üí° Key Insight: To evaluate means to find the value by substituting numbers for variables

Example 3: Real-World Variables

Problem: You buy pencils for $2 each. Write an expression for the total cost if you buy n pencils.

Step 1: Identify what changes ‚Üí number of pencils (n)
Step 2: Identify the pattern ‚Üí Cost = Price √ó Quantity
Step 3: Write the expression ‚Üí 2n or 2 √ó n

Answer: 2n dollars

üí° Key Insight: Variables let us write formulas that work for ANY number!

‚úèÔ∏è Practice with Variables!

Try these problems. Use hints if you get stuck!

Problem 1: Write an expression for "a number x decreased by 10"

Hint: "Decreased by" means subtraction. Start with x, then subtract 10.
Answer: x - 10

Problem 2: If y = 7, what is y + 15?

Hint: Replace y with 7, then add 15.
Answer: 22 (Because 7 + 15 = 22)

Problem 3: Write "the product of 6 and t" as an expression

Hint: "Product" means multiplication. Remember, you can write it as 6 √ó t or 6t.
Answer: 6t or 6 √ó t (both are correct!)

Problem 4: If a = 3, what is 5a - 8?

Hint: First substitute: 5(3) - 8. Then multiply before subtracting!
Answer: 7 (Because 5(3) = 15, and 15 - 8 = 7)

Problem 5: A movie ticket costs $12. Write an expression for the cost of n tickets.

Hint: Total cost = price per ticket √ó number of tickets
Answer: 12n dollars (or 12 √ó n)

Problem 6: Which expression means "triple a number plus 4"?
A) 3 + n + 4
B) 3n + 4
C) 3(n + 4)

Hint: "Triple" means multiply by 3. Do this FIRST, then add 4.
Answer: B) 3n + 4 ‚Äî Triple the number (3n), then add 4

üéØ Key Takeaways

  • Variables are letters that represent unknown or changing numbers
  • Expressions combine numbers, variables, and operations
  • When a number is next to a variable (like 5x), it means multiplication
  • To evaluate means to substitute a value and calculate
  • Variables make math more powerful‚Äîwe can write formulas that work for ANY number!
